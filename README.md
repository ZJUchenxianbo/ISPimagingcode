# ISPimagingcode
There are some codes generated by ai or myself which refer some books or papers for my own study about inverse scattering problem.

1.three_small_obstacles_joint_gn_random_centers : three objects, random centers,

2.three_small_obstacles_prior_sensitivity : to see the importance of the prior information.

3.three_small_obstacles_direct_imaging : direct imaging method

4.limited_aperture_direct_sampling_targets : fixed-frequency limited-aperture direct sampling for 1 small target, 3 small targets, and 1 large target. The observation aperture is controlled by the half-angle alpha.
